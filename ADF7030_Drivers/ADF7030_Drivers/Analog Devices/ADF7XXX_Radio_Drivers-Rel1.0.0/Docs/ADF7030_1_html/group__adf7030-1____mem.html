◆ adf7030_1__MEM_CheckByteAccess()

bool adf7030_1__MEM_CheckByteAccess ( uint32_t  nAddr)

Generic Function to check if byte rw operation is permitted.

Note
By default, all internal memory of the PHY Radio can be accessed at byte level. This includes ROM, RAM and Radio Peripheral Registers. APB Peripherals only support 32bits word access.
Parameters
[in]Addr32bit byte address location.
Returns
None
