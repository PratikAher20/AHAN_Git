◆ DISPLAY_BLOCK_CONTENT()

void DISPLAY_BLOCK_CONTENT ( DATA_BLOCK_DESCRIPTOR_Type *  pDATA__BLOCK,
void *  pOffset,
uint32_t  size 
)

Display Raw data from the host memory.

Note
If "pOffset" and "size" are NULL, the function will display the entire content of the host memory block pointed by pDATA__BLOCK. Else if pDATA__BLOCK is NULL and "pOffset" plus "size" are defined, this function will print "size" Bytes starting at location pointed by "pOffset".
Parameters
[in]pDATA__BLOCKPointer to the host data block structure.
[in]pOffsetPointer to desired host address location to start printing raw data.
[in]sizeNumber of bytes to be displayed
Returns
None
