Detailed Description

: This is a test file for adf7030_1 Radio driver development.

: This example continuously receives a packet and monitor baseband rx data on the adf7030_1. Signal available on PHY Radio GPIOs:

  • pin0 -> Baseband TRX Clock
  • pin1 -> Baseband TRX Data
  • pin2 -> External LNA framing
  • pin3 -> External IRQ0 - Toggles after each EOF_IRQn event
  • pin4 -> External IRQ1 - Toggles after each PREAMBLE_IRQn or PREAMBLE_GONE_IRQn or SYNCWORD_IRQn or LENGTH_IRQn or PAYLOAD_IRQn or PAYLOAD_BLOC_IRQn or CRC_CHK_IRQn or EOF_IRQn
  • pin5 -> Not used
  • pin6 -> Not used
  • pin7 -> Not used
